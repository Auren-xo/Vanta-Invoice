# Future Python API

The production-ready Python path for this app is FastAPI + SQLAlchemy + PostgreSQL. Python is not currently available on this machine, so the running application uses the included Node API and a local persistent JSON store. Install Python 3.12+ later, then add the FastAPI service here and point the frontend API base URL to it.

Recommended production database URL: `postgresql+psycopg://USER:PASSWORD@HOST:5432/vanta`
